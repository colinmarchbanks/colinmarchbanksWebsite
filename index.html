About me

Before CodeUp, I always knew that I wanted to get into the technology field, and went to college for a period of time trying to see where I belonged in that environment. I tried to give being a cable technician a shot but I didn't see the massive amount of driving or the lack of computer interaction as something that I wanted to make a career out of. At the soonest opportunity I enrolled in CodeUp to start a career behind a computer and I've been happy ever since.

As a CodeUp student, I always prided myself in the fact that I was able to keep up with the curriculum and go beyond the class by doing side projects and coding competitions while enrolled. I just never thought that simply doing what the rest of the class was doing was enough so I needed to find other projects to fill that gap in my life and to program more than the average student.

I see myself as an ambitious programmer and wanted to leave my mark on the world of technology, my goals in life involve touching many peoples lives with my programs and trying to make a positive change in the world we live in, with or without technology. Giving to charity and volunteering my time help me achieve this but I knew my impact will be much larger with technology at my beech and call.

Care Package

Winner of the 2019 Veteran's Code-a-thon in San Antonio.

This project was one conceived by me, and two other CodeUp students who saw a problem with our city and decided to take action to try to fix it. We noticed the Veteran's Code-a-thon had a challenge to try to combat veteran homelessness and we came up with the idea "Care Package", which is a way for the homeless to request supplies to their exact location or for other to request supplies to the location of a homeless person by using our app.

The application uses MapBox technology to get the current location of the user and asks the user if they would like a Care Package to that location or somewhere nearby. Then our backend gathers that data and is able to send out a volunteer to deliver the package to those in need. The theme of the care packages when it was conceived was hygiene and basic necessities but my team and I always saw this just as the beginning and are currently looking for ways to implement this project on a larger scale.

Weather Map

This is an application that can get a 3 day forecast anywhere.

This is an application that can get the three day weather forecast anywhere on the globe. It utilizes MapBox technologies as well as an API called DarkSkies to retrieve the weather. I like to style my work in a very visually appealing way without sacrificing information presented. This web-page follows this principle of mine to a key.

After I completed this project I felt like I had gotten quite adept at using and implementing APIs in my projects. This is also the project that I felt like defined my style, and I feel like that style is one that I'll continue to work on and build upon but I feel like it's one that works and is visually appealing.

Movie Application

An application shows various movie's details in a visually appealing way.

This application is a sort of movie database that I created with another CodeUp student. We originally had it only pull in movies that we had placed manually into the project, but we decided this was inefficient so we added functionality to add movies in real time and to pull information and poster of the movies from an API online.

In my opinion the movie posters are the best part of the overall project, and really add a visual appeal to it. Adding the movie description at that point was a necessity to tie the whole project together. The movie also pulls in the score from the API as well as allowing the user to input a personal score to the movie, as most of the time beauty is actually in the eye of the beholder.
